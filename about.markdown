#### Tools of choice

I use [Aquamacs](http://aquamacs.org/) on a [Macbook](http://www.apple.com/macbook) running [Mac OS X 10.6](http://www.apple.com/macosx/leopard/) to write code, and for servers I prefer [Debian](http://www.debian.org/) and [Ubuntu Server](http://www.ubuntu.com/). I use [Safari](http://www.apple.com/safari/) for browsing because [Firefox](http://www.mozilla.com/pt-BR/firefox/) is just too slow on Mac OS X. I use [ZSH](http://www.zsh.org/) as the default shell for terminal sessions.

My web framework of choice for small applications is [Sinatra](http://www.sinatrarb.com/), and most of the time I use [Python](http://www.python.org/) or [Ruby](http://www.ruby-lang.org/) as programming languages. I'm studying [Erlang](http://www.erlang.org/) now for writing server-like concurrent-based applications, and I'm learning more of [Javascript](http://en.wikipedia.org/wiki/JavaScript) by playing with [CouchDB](http://couchdb.apache.org/) views. While I have more experience with [prototype.js](http://www.prototypejs.org/), [JQuery](http://jquery.com/) is my Javascript framework of choice.
